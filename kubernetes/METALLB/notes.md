# 개요
* 자료조사

# 노트 정리
* bare metal환경에서는 load balancer 서비스 타입이 지원되지 않는 단점을 가지고 있다.

## 설치
* 외부에서 접근 가능한 IP대역 필요
  * 집인 경우 공유기 외부 IP
    * 공유기에서는 포트포워딩 설정이 되어 있어야하고 원룸 등일 경우 외부 접근이 허용되어야 한다.
