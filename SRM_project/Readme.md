# 설치
## python venv 생성과 활성화
```sh
python -m venv flask_venv
flask_venv\Script\activate.bat ; 윈도우
falsk_venv/Script/activate ; 리눅스
```

## 파이썬 패키지 설치
```sh
pip install -r requirement.txt
```

<br>

# 실행방법
```sh
python run.py
```
