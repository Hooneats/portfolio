# 개요
* vuetify 컴퍼넌트 소개

<br>

# v-app
> 참고자료: https://vuetifyjs.com/en/components/application/#api
* v-app은 vuetify를 사용하기 위해서 최상단에 **무조건 있어야하는** 컴퍼넌트

```html
<v-main>
</v-main>
```

<br>

# v-main
> 참고자료: https://vuetifyjs.com/en/features/layouts/
* content를 관리하는 컴퍼넌트

```html
<v-app>
  <v-main>
    Hello World
  </v-main>
</v-app>
```
